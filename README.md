# spring-cache-interceptors
:leaves: Cache interceptors for SpringFramework

## Development
### Test

```
$ ./gradlew clean check
```

### Upload Bintray

```
$ export BINTRAY_USER=username
$ export BINTRAY_KEY=apiKey
$ ./gradlew clean assemble bintrayUpload
```
